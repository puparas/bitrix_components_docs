# Bitrix User Help Documentation

Данная документация собрана с официального сайта Bitrix (https://dev.1c-bitrix.ru/user_help/) 06.06.25<br/>
Это документация по компонентам и модулям bitrix. Документация по api есть тут [/mosolovi/bitrix_full_docs](https://github.com/mosolovi/bitrix_full_docs)

## Directory Structure

- **-shope_settings.php.html**: Placeholder file indicating a missing resource.
- **additional/**: Additional resources and documentation.
- **clients/**: Documentation related to client management.
- **components/**: Detailed component documentation.
- **content/**: Content management documentation.
- **description_decisions/**: Descriptions of decision-making processes.
- **filter.md**: Documentation on complex filter logic.
- **index.md**: Main index of the documentation.
- **ites24/**: Documentation for Sites 24.
- **marketing/**: Marketing-related documentation.
- **marketplace/**: Marketplace documentation.
- **qintro.md**: Introduction to system requirements and implementation.
- **tatistic/**: Web analytics documentation.
- **terms.md**: Glossary of terms.
- **tore/**: Store-related documentation.
- **ttings/**: Settings documentation.
- **vice/**: Service-related documentation.

## Usage

To use the documentation, navigate through the directories and open the Markdown files in a Markdown viewer or editor. Each file contains detailed information on specific topics related to Bitrix.
