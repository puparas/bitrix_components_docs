# Список транспортных заявок

### Список заявок

Форма содержит список заявок.

| Поле | Описание |
| --- | --- |
| Колонка флажков | Выбор заявок, к которым предполагается применить какое-либо правило или действие. |
| Действия | Действия над заявкой:  * **Редактировать** - переход к [просмотру заявки](/user_help/store/sale/orders/sale_delivery_request_view.php); * **Удалить** - удаление заявки. |
| Идентификатор | Идентификатор заявки. |
| Дата и время создания | Дата и время создания транспортной заявки. |
| Служба доставки | Служба доставки, в которую отправлена заявка. |
| Внешний идентификатор | Внешний идентификатор заявки. |
| Номера заказов | Номера заказов, которые включены в заявку. |
| Идентификаторы отгрузок | Идентификаторы отгрузок, которые включены включены в заявку. |
