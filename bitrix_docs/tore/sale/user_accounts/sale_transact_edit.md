# Выполнение новой транзакции

Данная форма служит для проведения новой операции (транзакции) по счету клиента.

#### Контекстная панель

| Кнопка | Описание |
| --- | --- |
| Список транзакций | Переход на страницу со [списком](/user_help/store/sale/user_accounts/sale_transact_admin.php) транзакций. |

#### Поля формы

| Поле | Описание |
| --- | --- |
| \*Пользователь | Идентификатор пользователя, по счету которого выполняется операция. Идентификатор пользователя выбирается в списке, открываемом с помощью кнопки [**...**]. |
| \*Дата транзакции (DD.MM.YYYY HH:MI:SS) | Дата и время проведения операции.   Если значение поля указано для будущего периода, то операция будет выполнена при наступлении указанной даты и времени. |
| \*Сумма | Сумма операции. |
| Валюта | Валюта операции. |
| Тип записи | Тип выполняемой операции (транзакции). |
| Номер заказа (если транзакция относится к заказу) | Если транзакция выполняется в рамках какого-либо заказа, то в данном поле указывается номер этого заказа. |
| Заметки | Произвольный комментарий к выполняемой операции. |

\* - поля, обязательные для заполнения.
