# Список товаров

### Контекстная панель

| Кнопка | Описание |
| --- | --- |
| Создать | Доступны следующие варианты:  * **Товар** - создание простого товара без торговых предложений; * **Товар с торговыми предложениями** - создание товара с торговыми предложениями (действие доступно, если для инфоблока задан инфоблок торговых предложений); * **Услуга** - создание услуги; * **Комплект** - создание основного товара и комплекта к нему; * **Набор** - создание основного товара и набора к нему. |
| Добавить раздел | Переход к [форме создания](/user_help/store/catalog/products/cat_section_edit.php) нового раздела каталога. |
| Настроить | Переход к диалогу настройки внешнего вида отчетной формы. |
| Excel | Экспорт данных из отображаемой таблицы в MS Excel. |

### Список элементов

| Поле | Описание |
| --- | --- |
| Колонка флажков | Поле предназначено для выбора товаров и услуг, к которым предполагается применить какое-либо правило или действие. |
| Действия | Доступные действия:  * **Изменить** - переход к форме редактирования [товара](/user_help/store/catalog/products/cat_product_edit.php) или к форме редактирования [раздела](/user_help/store/catalog/products/cat_section_edit.php); * **Деактивировать/Активировать** - снятие/постановка флага активности для товара без захода в форму редактирования товара; * **Сбросить счетчик просмотров** - сброс счетчика просмотров товаров; * **Копировать** - копирование товара (копирование выполняется в тот же раздел каталога, где находится товар; впоследствии товар можно будет перенести в любой другой раздел этого каталога); * **Просмотреть** - переход к просмотру товара; * **Удалить** - удаление товара или раздела; * **Конвертировать в товар** - преобразование услуги в товар (для товаров будет необходимо формировать товарные накладные, а если вы работаете со складским учётом - указывать количество товара на складе и оформлять все складские документы); * **Конвертировать в услугу** - преобразование товара в услугу (после этого вы не сможете работать с этим товаром в складском учёте). |
| Тип товара | Указывается тип товара. |
| Название | Название товара/услуги или раздела. |
| Активность | Активность товара/услуги или раздела. |
| Сорт. | Относительный "вес", определяющий положение элемента (раздела) в списке. |
| Символьный код | Символьный код элемента (раздела). |
| Внешний код | Символьный код, используемый для связи элемента (раздела) с внешним источником данных. |
| Дата изм. | Дата и время последнего изменения элемента (раздела). |
| Кем изменена | Логин и имя пользователя, который совершил последние изменения. |
| Дата создания | Время и дата создания данной записи. |
| Кем создана | Логин и имя пользователя, который создал запись об элементе. |
| ID | Идентификатор товара. |
| Элементов | Количество элементов в разделе. |
| Подразделов | Количество подразделов в разделе. |
| Начало активности | Дата и время начала активности. |
| Окончание активности | Дата и время окончания активности. |
| Количество показов | Количество показов данного элемента на сайте. |
| Дата первого показа | Время и дата первого показа элемента на сайте. |
| Картинка для анонса | Изображение, указанное в анонсе товара или услуги. |
| Описание для анонса | Краткое описание товара или услуги для анонса. |
| Детальная картинка | Изображение, которое выводится в детальном описании товара или услуги. |
| Детальное описание | Детальное описание товара или услуги. |
| Теги | Теги товара или услуги. |
| [свойство] | Выводятся значения свойств, если они указаны. |
| Доступность | Признак доступности товара или услуги к покупке. |
| Доступное количество | Количество товара, доступное к покупке. |
| Коэф. | Коэффициент измерения товара. |
| Ед. измерения | Единица измерения товара или услуги. |
| Уменьшать количество при заказе | Количество товара уменьшается при заказе товара или нет. |
| Вес (грамм) | Вес товара в граммах. |
| НДС включен в цену | Признак включения НДС в цену товара или услуги. |
| Штрих-код | Штрих-код товара. |
| Закупочная цена | Величина закупочной цены товара. |
| [тип цены] | Цена товара или услуги данного типа цены. |
| Количество подписок | Количество подписок на товар или услугу. |

### Групповые действия

| Действие | Описание |
| --- | --- |
| активировать | Активировать элементы, которые на данный момент не активны. |
| деактивировать | Деактивировать активные элементы. |
| сбросить счётчик просмотров | Сброс счетчика просмотров элементов. |
| перенести в раздел | Выбор основного раздела элемента. |
| добавить привязку к разделу | Выбор дополнительных разделов, в которых будет отображаться элемент. |
| заблокировать/разблокировать | Опция доступна при включенном Документообороте    Настройка документооборота на последовательную работу над информационными блоками включает в себя шаги:  1. Настройка прав модуля **Управление структурой**. 2. Настройки разделов, участвующих в документообороте. 3. Создание и настройка необходимых статусов для документов.    [Подробнее](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=41&LESSON_ID=5049)... и позволяет разрешить/запретить работу с элементом. |
| изменить статус | Опция доступна при включенном Документообороте,    Настройка документооборота на последовательную работу над информационными блоками включает в себя шаги:  1. Настройка прав модуля **Управление структурой**. 2. Настройки разделов, участвующих в документообороте. 3. Создание и настройка необходимых статусов для документов.    [Подробнее](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=41&LESSON_ID=5049)... позволяет изменить текущий статус элемента. |
| параметр товаров | В качестве Параметра товара    Групповое действие **Параметр товара** доступно с версии **19.0.0** модуля **Информационные блоки** и с **19.0.0** модуля **Торговый каталог**. можно указать одну из следующих позиций:  * **Вес** – указывается вес торгового предложения в граммах; * **\*Режим количественного учета** – включается   режим количественного учёта      **Количественный учет** - режим работы с товарами в интернет-магазине на базе *«1С-Битрикс: Управление сайтом»*, позволяющий изменять количество товара как вручную, так и автоматически после оформления заказов покупателями.     [Подробнее](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=42&LESSON_ID=5298)...   для отмеченных торговых предложений; * **\*Разрешение покупки при отсутствии товара** – при отмеченной опции товар будет по умолчанию доступен к покупке даже при нулевом количестве; * **\*Доступное количество** – количество товара в данный момент; * **\*Закупочная цена** – величина закупочной цены товара; * **НДС включен в цену** – проставляется признак включения   НДС      **Налог на добавленную стоимость (НДС)** — косвенный налог, форма изъятия в бюджет государства части добавленной стоимости, которая создаётся на всех стадиях процесса производства товаров, работ и услуг и вносится в бюджет по мере реализации.     [Подробнее](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=42&LESSON_ID=3458)...   в цену товара или услуги; * **Идентификатор НДС** – выбирается одна из ранее созданных   ставок НДС;      По умолчанию в Интернет-магазин уже включены **Ставки НДС**: **0%** и **18%**. Ставка НДС может быть разной в зависимости от товара. Создадим и применим новую **Ставку НДС**.     [Подробнее](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=42&LESSON_ID=3458)... * **Разрешение подписки на товар** – при отмеченной опции для элемента будет показываться кнопка   подписки,      Начиная с версии **16.5.3** модуля **Торговый каталог**, в системе доступен обновленный функционал подписок на товары, временно отсутствующие в продаже.        Для отсутствующего в магазине товара кнопка подписки будет отображаться в том случае, если в системе для него выполнены следующие настройки:       если он недоступен к покупке; * **Единица измерения** – выбирается из списка созданных ранее   единиц измерений;      Настройка единиц измерения для интернет-магазина подробно описана в видео-уроке. В РФ в соответствии с Федеральным законом от 31 декабря 2017 г. № 487-ФЗ товары подлежат обязательной маркировке, согласно утвержденному перечню. Для редакции **Малый бизнес** наложено ограничение - возможно использование только **одного** склада.
