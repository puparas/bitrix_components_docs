# Настройка sitemap

### Список настроек

| Колонка | Описание |
| --- | --- |
| Колонка флажков | Выбор настроек, к которым предполагается применить какое-либо действие. |
| Меню действий | доступные действия:  * **Редактировать** - переход к [редактированию](/user_help/marketing/seo/seo_sitemap_edit.php) настройки генерации карты сайта; * **Запустить** - запуск генерации карты сайта sitemap.xml; * **Удалить** - удаление настройки. |
| ID | Идентификатор настройки. |
| Изменено | Дата и время последнего изменения настройки или последнего запуска генерации карты сайта. |
| Название | Название настройки карты сайта. |
| Сайт | Идентификатор и название сайта, для которого составлена настройка. |
| Последний запуск | Дата и время последнего запуска генерации карты сайта. |
| кнопка *Запустить* | Запуск генерации карты сайта sitemap.xml. |
