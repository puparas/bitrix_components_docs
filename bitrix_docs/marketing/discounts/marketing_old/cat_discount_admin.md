# Скидки на товар

### Контекстная панель

| Кнопка | Описание |
| --- | --- |
| Новая скидка | Переход к форме [добавления](/user_help/marketing/discounts/marketing_old/cat_discount_edit.php) новой скидки. |
| Настроить | Позволяет перейти к диалогу настройки внешнего вида отчетной формы. |
| Excel | Экспортирует данные из отображаемой таблицы в **MS Excel**. |

### Список скидок

| Поле | Описание |
| --- | --- |
| Колонка флажков | Поле предназначено для выбора скидок, к которым предполагается применить какое-либо правило или действие. |
| Действия | Действия над скидкой:  * **Изменить параметры скидки** - [редактирование](/user_help/marketing/discounts/marketing_old/cat_discount_edit.php) скидки; * **Копировать скидку** - копирование скидки. * **Удалить скидку** - удаление скидки. |
| ID | Идентификатор скидки. |
| Сайт | Название сайта(ов), к товарам которого может быть применена данная скидка. |
| Активна с | Дата начала активности скидки. |
| по | Дата окончания активности скидки. |
| Акт. | Признак активности скидки. Если скидка неактивна, то она не может быть применена. |
| Приоритет применимости | Число приоритета применимости скидки. |
| Название | Название скидки. |
| Сорт. | Относительный "вес" скидки. Используется для вывода скидки в общем списке. |
| Величина | Размер предоставляемой скидки. |
| Продл. | Признак использования скидки для продления подписки на контент сайта. |
| Кем изменена | Логин пользователя, последний раз изменявшего скидку. |
| Дата изменения | Дата последнего изменения скидки. |

**Примечание:** сумма скидки вычисляется от конечной цены с уже начисленным НДС.
