# Старт

### Недавние выпуски

Выводится список недавних выпусков, а также есть возможность **Создать новую рассылку**.

| Поле | Описание |
| --- | --- |
| Создать новую рассылку | При нажатии этой кнопки откроется мастер создания новой рассылки.   Второй вариант создания рассылки - в разделе **Рассылки** (Маркетинг > Email-маркетинг > Рассылки) нажать кнопку **Создать рассылку**. |
