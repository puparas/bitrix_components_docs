# Поиск по каталогу

### Описание **catalog.search**

Компонент выводит результаты поиска по элементам каталога с указанным набором свойств, цен и т.д.. Компонент стандартный, входит в дистрибутив модуля и содержит два шаблона: **.default** и **bootstrap\_v4**.

В визуальном редакторе компонент расположен по пути *Контент > Каталог > Поиск по каталогу*.

Компонент относится к модулю [Информационные блоки](/user_help/content/iblock/index.php).
