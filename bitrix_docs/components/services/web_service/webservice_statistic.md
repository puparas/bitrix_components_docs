# Веб-сервисы модуля статистики

### Описание **webservice.statistic**

Веб-сервис, отдающий статистические данные по сайту, например, для гаджета статистики для боковой панели OC **Windows Vista** (*/bitrix/components/bitrix/webservice.statistic/distr*). Компонент стандартный и входит в состав дистрибутива.

В визуальном редакторе компонент находится в *Компоненты > Сервисы > Веб-Сервис*.

Компонент относится к модулю [Веб-сервис](/user_help/settings/webservice/index.php).

### Пример вызова

```
<?$APPLICATION->IncludeComponent("bitrix:webservice.statistic","",Array(
   ),
);?>

```
