# Получение данных в формате metaWeblog

Данный компонент осуществляет импорт в формате **MetaWeblog**. Позволяет добавлять сообщения из **Microsoft Word** и **Microsoft Live Writer** на сайт в блоги пользователя и группы социальной сети.

Компонент настроек не имеет, его только необходимо разместить на удобной вам странице. Затем в настройках учетной записи **Microsoft Word** или **Microsoft Live Writer** необходимо задать адрес к данной странице и указать ваши параметры доступа к сайту.

В структуре визуального редактора компонент расположен по пути *Общение > Блоги > Получение данных в формате metaWeblog*.

Компонент относится к модулю [Блоги](/user_help/service/blogs/index.php).

По умолчанию компонент добавлен в комплексный компонент [Блоги](/user_help/components/obschenie/blogs/blog.php).
