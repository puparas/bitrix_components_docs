# Бизнес-процесс

Подраздел служит для автоматизации бизнес-процессов организации и включает в себя следующие компоненты:

| **Компоненты** | | |
| --- | --- | --- |
| **Название** | **Имя в пространстве имен bitrix** | **Описание** |
| [Бизнес-процесс (комплексный компонент)](/user_help/components/obschenie/buziness_process/bizproc_wizards.php) | **bizproc.wizards** | Служит для создания полноценного публичного интерфейса для работы с бизнес-процессами организации. |
