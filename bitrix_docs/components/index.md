Под опубликованными документами можно организовать обсуждение. |
| [Задачи 2.0 (КП)](/user_help/components/content/tasks/index.php) | **tasks** | Предназначен для управления задачами на портале. |

| [Фотогалерея](/user_help/components/content/photogallery/index.php) | **photo** | Подраздел предназначен для размещения на сайте полнофункциональной фотогалереи. |
| [Фотогалерея 2.0](/user_help/components/content/photogallery2/index.php) | **photogallery** | Подраздел предназначен для организации полнофункциональной фотогалереи с гибко настраиваемыми параметрами, в т.ч. многопользовательской. |
| [Каталог](/user_help/components/content/catalog/index.php) | **catalog** | Служит для организации и размещения на сайте полнофункционального каталога товаров. |

| [Highload инфоблоки](/user_help/components/content/highload/index.php) | **highloadblock** | Служит для работы с произвольными наборами данных в условиях высоких нагрузок. |

| [Wiki](/user_help/components/content/wiki/index.php) | **wiki** | Позволяет организовать коллективную работу над контентом страницы. |
| [Валюты](/user_help/components/content/currency/index.php) | **currency** | Позволяет управлять валютами и их курсами. |
| [Добавление элементов](/user_help/components/content/adding/index.php) | **iblock.element.add** | Предназначен для работы с элементами инфоблока. |
| [Инфоблоки](/user_help/components/content/infoblocks/index.php) | **iblock** | Служит для управления и каталогизирования различных типов (блоков) однородной информации. |
| [Календарь событий](/user_help/components/content/calendar/index.php) | **calendar** | Дает возможность просматривать график предстоящих и прошедших событий сотрудников. |
| [Карта сайта](/user_help/components/content/sitemap/index.php) | **main.map** | Осуществляет показ карты сайта. |

| [Обмен с 1С](/user_help/components/content/1c_exchange/index.php) |  | Предназначен для обмена данными с 1С в формате CommerceML v2. |
