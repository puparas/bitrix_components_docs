# Просмотр сделки

### Описание **crm.deal.show**

Одностраничный компонент, осуществляющий вывод детального описания сделки. Компонент стандартный и входит в дистрибутив модуля.

В визуальном редакторе компонент расположен по пути *CRM > Сделки > Просмотр сделки*.

### Параметры

|  |  |  |
| --- | --- | --- |
| **Поле** | **Параметр** | **Описание** |
| **Основные параметры** | | |
| ID сделки | **ELEMENT\_ID** | Поле содержит код, в котором передается идентификатор сделки. |

### Пример вызова

```
<?$
APPLICATION->IncludeComponent(
"bitrix:crm.deal.show",
	"",
	Array(
		"ELEMENT_ID" => $_REQUEST["deal_id"]
	)
);?>

```
