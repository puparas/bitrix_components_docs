# Комментарии

### Контекстная панель

| Кнопка | Описание |
| --- | --- |
| Настроить | Переход к диалогу настройки внешнего вида отчетной формы. |
| Excel | Экспорт данных из таблицы в MS Excel. |

### Список комментариев

| Поле | Описание |
| --- | --- |
| Колонка флажков | Поле для выбора комментариев, к которым предполагается применить какое-либо действие. |
| Действия | Действия над комментариями:  * **Скрыть** - присвоение комментарию статуса "неопубликован"; * **Удалить** - удаление комментария. |
| ID | Идентификатор комментария. |
| Опубликован | Статус комментария. |
| Дата создания | Дата создания комментария. |
| Автор | Автор комментария. |
| Комментарий | Текст комментария. |
| Сообщение | Сообщение, к которому был написан комментарий. |
| Блог | Идентификатор и имя блога, в котором был помещен комментарий. |
| IP | IP компьютера, с которого был отправлен комментарий. Нажав на [Стоп-лист](/user_help/statistic/visitors/stoplist_list.php) можно ограничить доступ этого посетителя на сайт |
