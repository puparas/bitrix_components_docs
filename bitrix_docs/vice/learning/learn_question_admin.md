# Вопросы тестов

### Контекстная панель

| Кнопка | Описание |
| --- | --- |
| Добавить вопрос | Служит для перехода к форме создания [нового вопроса](/user_help/service/learning/learn_question_edit.php). |
| Настроить | Переход к диалогу настройки внешнего вида отчетной формы. |
| Excel | Экспорт данных из отображаемой таблицы в MS Excel. |

### Список вопросов

| Колонка | Описание |
| --- | --- |
| Колонка флажков | Выбор вопросов, к которым предполагается применить какое-либо действие. |
| Меню действий | доступные действия:  * **Изменить** - [отредактировать](/user_help/service/learning/learn_question_edit.php) вопрос; * **Удалить** - удалить вопрос. |
| ID | Идентификатор вопроса. |
| Акт. | Признак активности вопроса. |
| Дата изм. | Дата и время последней модификации вопроса. |
| Название | Текст самого вопроса. |
| Сорт. | Вес сортировки вопроса; применяется при выводе вопроса. |
| Самопров. | Принадлежность вопроса: для самопроверки или не для самопроверки. |
| Об. прав. | Обязателен правильный ответ. |
| Тип | Указывается тип вопроса: одиночный или множественный выбор вариантов ответа. |
| Балл | Указывается балл, начисляемый за правильный ответ. |
| Кол-во ответов (правильных/всего) | Указывается количество **правильных** из **всех** ответов. Ссылки на количествах ведут на [страницы результатов теста](/user_help/service/learning/learn_test_result_admin.php) с фильтром на соответствующие ответы. |
