# Создание и редактирование вопроса

### Вопрос

| Поле | Описание |
| --- | --- |
| ID | Идентификатор вопроса.     Поле отображается только в режиме редактирования существующего вопроса. |
| Изменена | Дата и время создания или изменения вопроса.     Поле отображается только в режиме редактирования существующего вопроса. |
| Запись активна | Признак активности вопроса. Если это поле отмечено, то вопрос доступен для показа в публичной части. |
| Тип вопроса | Выбор типа вопроса:    * **одиночный выбор** - только один вариант ответа; * **множественный выбор** - возможен несколько вариантов ответа; * **сортировка** - необходимо расставить ответы в каком-либо порядке; * **текстовый ответ** - возможен только текстовый ответ, набираемый отвечающим на вопрос пользователем (поле для ввода текста появится на месте ответов). |
| Урок | Урок, к которому относится вопрос.    Для выбора другого урока, к которому должен быть привязан вопрос, служит ссылка **сменить**. Откроется окно со списком глав и уроков курса: для выбора необходимого урока (или главы) кликнете по нему. |
| \*Название | Текст самого вопроса. |
| Индекс сортировки | Вес сортировки вопроса; применяется при выводе вопроса. |
| Балл | Количество баллов, которые зачисляются пользователю в случае верного ответа на данный вопрос. |
| Для самопроверки | Признак принадлежности вопроса: для самопроверки или нет.    **Опция доступна, только если выбран тип вопроса - одиночный выбор, множественный выбор, сортировка.** |
| Обязателен правильный ответ | При отмеченной опции если на данный вопрос не будет дан правильный ответ, то тест не будет сдан при любом набранном количестве баллов. |
| Сообщение при неправильном ответе | Указывается сообщение при неправильном ответе на вопрос во время прохождения теста. Сообщение будет выводится при включенной опции **Показывать ошибки** (*Сервисы > Обучение > Курсы > имя\_курса > Тесты*) закладки "Настройки". |
| Уведомлять преподавателя об ответе | При отмеченной опции преподавателю (пользователю, создавшему урок) отправляется письмо с ответом ученика.   **Примечание:** при необходимости можно добавить получателей, которым будет отправляться данное письмо. Оно отправляется по событию **NEW\_LEARNING\_TEXT\_ANSWER** (*Настройки > Настройки продукта > Почтовые события > Типы почтовых событий*). |
| Заметка | Произвольный комментарий к вопросу, не видим для пользователя, отвечающего на вопрос. |
| Изображение | К тексту вопроса можно добавить изображение с помощью кнопки **Добавить файл**. |

\* - поля, обязательные для заполнения.

### Ответы

Задание вариантов ответов на вопрос.

| Поле | Описание |
| --- | --- |
| ID | Идентификатор вопроса.   Это поле заполнено только в режиме редактирования вопроса. |
| Прав. | С помощью данного поля необходимо отметить один или несколько верных вариантов ответа в зависимости от типа вопроса. |
| Ответ | Текст ответа на вопрос. |
| Сорт. | Порядок вывода ответов в вопросе при выключенной опции [Случайный порядок ответов](/user_help/service/learning/learn_test_edit.php#test) в данном **Тесте**. |
| Удал. | Удаление вариантов ответа возможно только в режиме редактирования. Для этого необходимо отметить флаг тот вариант ответа, который вы хотите удалить. |

### Описание

Создание детального описания вопроса, которое может содержать как текст, так и изображения.

| Поле | Описание |
| --- | --- |
| Переключатель режима ввода текста | Задается режим ввода текста вопроса: **Text**, **HTML** или **Визуальный редактор**.    В текстовом режиме отсутствует возможность форматирования текста и вставки рисунков. Режим HTML позволяет использовать любое форматирование сообщения с помощью HTML-тегов. Визуальный редактор позволяет редактировать контент так, как он непосредственно будет выглядеть. |

### Доп. поля

На данной закладке отображаются созданные в системе дополнительные пользовательские поля для вопросов модуля **Обучение**. К созданию нового пользовательского поля можно перейти по ссылке [Добавить пользовательское поле](/user_help/settings/settings/userfield_admin.php).
