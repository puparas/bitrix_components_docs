# Расписания

Страница **Расписания** (*Сервисы > Техподдержка > Расписания > Расписания*) служит для создания и настройки расписаний работы техподдержки. Расписания используются при настройке уровней техподдержки (SLA).

  

### Фильтр

Форма фильтра используется для выбора из списка расписаний в соответствии с указанными условиями. Нижеследующая таблица описывает параметры, по которым может выполняться поиск.

| Параметр | Описание |
| --- | --- |
| Имя | Поиск по названию расписания. Поле допускает использование [специальных логических выражений](https://dev.1c-bitrix.ru/api_help/main/general/filter.php). |

Чтобы отобрать расписания по заданным критериям поиска, нажмите кнопку **Найти**. Для отображения всех расписаний нажмите кнопку **Отменить**.

### Контекстная панель

| Кнопка | Описание |
| --- | --- |
| Добавить | Добавление [нового расписания](/user_help/service/support/ticket_timetable/ticket_timetable_edit.php). |
| Настроить | Переход к диалогу настройки внешнего вида отчетной формы. |
| Excel | Экспорт данных из отображаемой таблицы в MS Excel. |

### Список расписаний

| Поле | Описание |
| --- | --- |
| Колонка флажков | Поле предназначено для выбора расписаний, к которым предполагается применить какое-либо правило или действие. |
| Действия | Действия, которые можно произвести над расписанием:  * **Изменить** - переход к [форме редактирования параметров расписания](/user_help/service/support/ticket_timetable/ticket_timetable_edit.php); * **Удалить** - удаление расписания. |
| ID | Идентификатор расписания. |
| Название | Название расписания. |
| Описание | Описание расписания. |
