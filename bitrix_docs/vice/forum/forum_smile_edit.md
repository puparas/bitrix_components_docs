# Создание и редактирование смайлика (до v.15.5.2)

**Внимание!** Начиная с версии 15.5.2 в модуле **Форум** используются [смайлы ядра системы](/user_help/service/smile/smile_gallery.php).

Форма предназначена для создания новых и редактирования уже существующих смайликов.

  

### Контекстная панель

| Кнопка | Описание |
| --- | --- |
| Список смайликов | Переход на страницу со [списком](/user_help/service/forum/forum_smile.php) смайликов. |
| Добавить новый смайлик | Добавление нового смайлика. Кнопка отображается при редактировании существующего смайлика. |
| Удалить смайлик | Удаление редактируемого смайлика. |

### Форма редактирования

| Поле | Описание |
| --- | --- |
| Код | Уникальный системный код смайлика. Поле отображается при редактировании смайлика. |
| Сортировка | Вес сортировки смайлика. Определяет порядок вывода смайликов в списке. |
| Тип смайлика | Укажите тип смайлика: смайлик или иконка. |
| Написание смайлика | Символьный эквивалент изображения. Возможно перечисление нескольких вариантов через пробел. |
| Изображение смайлика | Изображение смайлика или иконки. Для загрузки изображения служит кнопка **Обзор**. Рекомендуемый размер смайликов: 16x16, иконок: 15x15. |
| [Язык] | |
| \*Название смайлика | Название изображения, показываемое в публичной части. Название указывается для каждого языка, используемого в системе. |

\* - Поля, обязательные для заполнения.
