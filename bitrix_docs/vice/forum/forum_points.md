# Звания

Страница **Звания** (*Сервисы > Форумы > Звания*) содержит список созданных званий, а также позволяет перейти к созданию новых и изменению существующих званий.

Звания посетителей устанавливаются в зависимости от количества набранных баллов. Пользователь может получить баллы как за количество сообщений, созданных на форуме, так в результате голосования других посетителей форума.

**Примечание:** для использования данного функционала следует включить опцию **Включить бонусную систему** в глобальных [настройках модуля](/user_help/service/forum/settings.php) форума. Если опция **Включить бонусную систему** неактивна, то звания будут устанавливаться на основе [рейтингов](/user_help/service/rating/rating_list.php).

  

### Контекстная панель

| Кнопка | Описание |
| --- | --- |
| Новое звание | Переход к форме [создания](/user_help/service/forum/forum_points_edit.php) нового звания. |
| Настроить | Позволяет перейти к диалогу настройки внешнего вида отчетной формы. |
| Excel | Экспорт данных из отображаемой таблицы в MS Excel. |

### Список званий

| Поле | Описание |
| --- | --- |
| Колонка флажков | Выбор званий, к которым предполагается применить какое-либо правило или действие. |
| Действия | Действия, которые можно произвести над званием:  * **Изменить параметры звания** - [редактирование](/user_help/service/forum/forum_points_edit.php) звания; * **Удалить звание** - удаление звания. |
| ID | Идентификатор звания в системе. |
| Название | Название звания, как оно отображается на форуме. |
| Необходимо баллов | Количество баллов, требуемое для получения этого звания. **Примечание:**Доступно при включенной опции **Включить бонусную систему** в настройках модуля **Форум** (*Настройки > Настройки продукта > Настройки модулей > Форум*). |
| Голосов | Количество голосов, требуемое для получения этого звания. **Примечание:**Доступно при включенной опции **Включить бонусную систему** в настройках модуля **Форум** (*Настройки > Настройки продукта > Настройки модулей > Форум*). |
| Необходимо нормированных голосов | Количество нормированных голосов, требуемое для получения этого звания. **Примечание:**Доступно при выключенной опции **Включить бонусную систему** в настройках модуля **Форум** (*Настройки > Настройки продукта > Настройки модулей > Форум*). |
