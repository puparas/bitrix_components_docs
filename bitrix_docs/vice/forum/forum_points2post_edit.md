# Создание и редактирование баллов за сообщение

Форма служит как для создания новых, так и для редактирования существующих записей с количеством баллов за сообщение.

  

### Контекстная панель

| Кнопка | Описание |
| --- | --- |
| Список записей | Переход на страницу со [списком записей о количестве баллов](/user_help/service/forum/forum_points2post.php), присваиваемых за сообщение. |
| Добавить новую запись | Переход к форме создания новой записи. Кнопка отображается при редактировании существующей записи. |
| Удалить запись | Удаление редактируемой записи. |

### Форма редактирования

| Поле | Описание |
| --- | --- |
| ID | Идентификатор записи в системе. Отображается только при редактировании записи. |
| \*Необходимо сообщений | Число сообщений, начиная с которого за каждое сообщение будет начисляться указанное в записи количество баллов. |
| Баллов за сообщение | Количество баллов, начисляемое за сообщение, при достижении указанного в предыдущем поле числа сообщений. |

\* - Поле, обязательное для заполнения.
