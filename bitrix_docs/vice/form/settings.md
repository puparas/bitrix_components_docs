# Веб-формы. Настройки модуля

Закладки

### Связь с CRM

| Поле | Описание |
| --- | --- |
| [привязанные\_порталы] | Для каждого привязанного портала указывается его название, путь к настройкам CRM портала и информация о соединении с порталом. |
| кнопка **Добавить** | Кнопка служит для добавления новой связи. При ее нажатии открывается форма, в которой указывается:  * название привязанного портала * адрес сервера портала * адрес страницы с компонентом интеграции относительного корневого каталога сервера (по умолчанию адрес уже задан) * логин и пароль сотрудника портала, который будет выполнять обработку лидов |

### Доступ

| Поле | Описание |
| --- | --- |
| По умолчанию | Управление уровнем доступа групп пользователей, для которых установлено право "по умолчанию". |
| [группа\_пользователей] | Управление правом доступа для конкретной группы. Возможно назначение следующих прав доступа к модулю **Веб-формы**:  * **[D] доступ закрыт** — доступ к меню и всем административным файлам модуля закрыт * **[R] доступ открыт** — доступ к меню и административным файлам модуля открыт. Дальнейшие права доступа к формам регламентируются правами, заданными в каждой конкретной форме * **[W] полный доступ** — полный доступ к ресурсам модуля, а также право на создание новых форм и настройку всех форм |
