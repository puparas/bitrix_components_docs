# Веб-мессенджер. Настройки модуля

**Внимание**: в *1С-Битрикс: Управление сайтом* поддержка чата (модуля im) прекращена.

### Описание модуля

Обратите внимание:

* Без настроенного модуля [Push&Pull](/user_help/settings/pull/settings.php) и **сервера очередей** не будут работать следующие функции: групповой чат, редактирование сообщений, передача файлов, телефония (внешние и внутренние звонки на портале);
* Без модуля [Телефония (voximplant)](/user_help/settings/voximplant/index.php) не будет работать телефония (внешние звонки с портала);
* Опция **Использовать для видеозвонков SFU сервер Voximplant** требует установленный модуль [Телефония (voximplant)](/user_help/settings/voximplant/index.php) и **активную лицензию**;
* Без модуля **Disk** не будет работать передача файлов.

Для улучшения быстродействия модуля **Веб-мессенджер** версии **17.0.4 и выше** необходимо:

* Обновить вашу БД MySQL до версии 5.6.
* После этого [в командной строке базы данных](/user_help/settings/utilities/sql.php) выполнить следующий запрос:

  ```
  
  CREATE fulltext index IXF_IM_MESS_1 on b_im_message

  ```

Если вы устанавливали продукт на сервер с уже установленной БД MySQL 5.6 и выше, то вам ничего делать не нужно.

### Настройки

Форма настроек модуля **Веб-мессенджер** (*Настройки > Настройки продукта > Настройки модулей > Веб-мессенджер*) предназначена для задания параметров модуля.

| Поле | Описание |
| --- | --- |
| Использовать для видеозвонков SFU сервер Voximplant | Если опция включена, то медиа-поток звонка пойдет через сервера VoxImplant и будет разрешено подключение до 48 участников. Если не стоит - медиа-поток идет напрямую между участниками, разрешено до 4-х участников. По умолчанию не стоит.   Доступно только в "Битрикс24 в коробке" |
| Использовать свой TURN сервер видеозвонков | При отмеченной опции указываются настройки собственного сервера видеозвонков.   По умолчанию:  * **Адрес сервера**:`turn.calls.bitrix24.com` * **Адрес сервера для Firefox**:`54.217.240.163` * **Логин**:`bitrix` * **Пароль**:`bitrix` |
| Скрывать отсутствующих пользователей | При отмеченной опции отсутствующие пользователи по умолчанию будут скрыты в контакт-листе. Каждый пользователь может включать или выключать данную опцию в [настройках Бизнес-чата](http://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=71&LESSON_ID=7487) в публичной части. |
| Скрывать группы пользователей | При отмеченной опции группы по умочланию будут скрыты в контакт-листе. Каждый пользователь может включать или выключать данную опцию в [настройках Бизнес-чата](http://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=71&LESSON_ID=7487) в публичной части. |
| Загружать последние сообщения | При отмеченной опции по умочланию будут загружены последние сообщения в диалогах. Каждый пользователь может включать или выключать данную опцию в [настройках Бизнес-чата](http://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=71&LESSON_ID=7487) в публичной части. |
| Загружать историю уведомлений | При отмеченной опции по умочланию будет загружена история уведомлений. Каждый пользователь может включиать или выключать данную опцию в [настройках Бизнес-чата](http://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=71&LESSON_ID=7487) в публичной части. |
| Сочетание клавиш для отправки сообщения | Указывается сочетание клавиш для отправки сообщений: **Enter** или **Ctrl + Enter** по умолчанию. |
| Расположение панели мессенджера | Указывается расположение панели мессенджера по умолчанию. **Примечание**: В шаблоне **Битрикс24** портала панель мессенджера скрыта и принудительно расположена справа по всей высоте экрана. |
| Принимать сообщения от | Указывается, от кого разрешено получать сообщения по умолчанию.    Доступно только в "1С-Битрикс: Управление сайтом" |
| Принимать звонки от | Указывается, от кого разрешено получать звонки по умолчанию.    Доступно только в "1С-Битрикс: Управление сайтом" |
| Приглашать в чат могут | Указывается, от кого разрешено получать приглашение в чат по умолчанию.   Доступно только в "1С-Битрикс: Управление сайтом" |
| Отображать контакт при поиске для | Указывается, для кого разрешено отображать свой контакт при поиске.   Доступно только в "1С-Битрикс: Управление сайтом" |
| При приглашении пользователя в чат, ему доступна история сообщений | Указывается, как показывать историю сообщений группового чата при приглашении в него нового участника:  * **с момента входа** - ранние сообщения участников группового чата приглашенному участнику будут не видны; * **с первого сообщения** - будут видны все сообщения с момента создания чата. |
| Включить цветовую схему для чатов и пользователей | При отмеченной опции будут включены [Цветные чаты](http://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=71&LESSON_ID=7491#color_chats). |
| Включить открытые чаты | При отмеченной опции будут включены [Открытые чаты](http://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=71&LESSON_ID=7491#chats), в том числе и **Общий чат**. |
| Включить предварительную загрузку всех пользователей портала для быстрого поиска | Отключение опции рекомендуется для компаний с сотрудниками выше 300 человек. Это позволит снизить нагрузку на веб-клиент: при в отключенном состоянии будут подгружаться в мессенджер не все пользователи, а только те с которыми сотрудник переписывается или находит в поиске.   Доступно только в "Битрикс24 в коробке" |
| Включить автоматическое сообщение о новом сотруднике в общий чат | При отмеченной опции в Общий чат будет приходить сообщение о принятии на работу нового сотрудника.   Доступно только в "Битрикс24 в коробке" |
| Включить автоматическое сообщение об увольнении сотрудника в общий чат | При отмеченной опции в Общий чат будет приходить сообщение об увольнении сотрудника.   Доступно только в "Битрикс24 в коробке" |
| Настройки по сайтам или порталам | |
| Формат отображения имени в списке контактов | Указывается формат отображения имени в списке контактов Бизнес-чата. Настройка доступна для разных сайтов. |
| Шаблон пути к странице пользователя | Указывается шаблон пути к странице пользователя. По умолчанию: `/club/user/#user_id#/`.    Доступно только в "1С-Битрикс: Управление сайтом" |
