# Контроллер. Настройки модуля

### Доступ

Закладка позволяет настроить права на доступ к модулю для всех имеющихся в системе групп пользователей.

| Поле | Описание |
| --- | --- |
| По умолчанию | Уровень доступа групп пользователей, для которых установлено право "по умолчанию". |
| [группа\_пользователей] | Изменение настроек доступа для конкретной группы. Возможно назначение следующих прав доступа к модулю **Контроллер**:  * **[D] Нет доступа** - запрет на доступ; * **[L] Авторизация** - разрешается выполнить вход на подчиненных сайтах; * **[R] Чтение** - разрешается только просмотр файлов и параметров модуля; * **[T] Добавление** - добавление новых сайтов, позволяет безопасно вводить имя-пароль на клиентских машинах при присоединении; * **[V] Управление** - управление сайтами, т.е. их добавление, обновление, отсоединение, административный вход на них, просмотр журнала работы. Нельзя удалять сайты и изменять групповые настройки; * **[W] Полный доступ** - полный доступ без каких-либо ограничений. |
