# Социальная сеть. Настройки модуля

Вкладки

Форма настроек модуля **Социальная сеть** (*Настройки > Настройки продукта > Настройки модулей > Социальная сеть*) предназначена для задания общесистемных параметров модуля.

  

### Настройки модуля социальной сети

| Поле | Описание |
| --- | --- |
| Глобальные настройки модуля социальной сети | |
| Следить за всеми сообщениями Живой ленты по умолчанию | При отмеченной опции в социальной сети каждое новое событие (комментарий, сообщение, загрузка файла, вступление в группу и т.п) будет "поднимать" данное событие наверх Живой ленты. Чтобы не следить за событием, пользователь сам решает следить или нет с помощью кнопки **Больше не следить** под сообщением Живой ленты. При отключенной опции, чтобы следить за событием, необходимо нажать кнопку **Следить** под нужным сообщением Живой ленты. |
| Разрешать адресовать "Всем" сообщения в Живой ленте | При отмеченной опции появляется возможность указать пользователей или группы пользователей, которым разрешено адресовать сообщения для всех пользователей портала. |
| Отображать адресацию "Всем" по умолчанию | При отмеченной опции будет включена адресация "Всем" по умолчанию при создании сообщения. **Примечание**: Опция появляется при включенной опции "Разрешать адресовать "Всем" сообщения в Живой ленте". |
| Отображать всех почтовых получателей | При отмеченной опции будет включено отображение всех почтовых получателей. |
| Отображать в Живой ленте по умолчанию только мои записи, записи моих групп и адресованные лично мне | При отмеченной опции в социальной сети включается режим, при котором выбираются только записи, относящиеся к рабочим группам пользователя, а также его собственные или адресованные лично ему. Режим включается/выключается как глобально, так и каждым пользователем в Живой ленте (пункт **Только мои группы** в меню **Все сообщения**).  **Примечание**: Опция доступна только для «1С-Битрикс: Управление сайтом». |

| Шаблон пути для группы | Указывается шаблон пути к странице просмотра рабочих групп. |
| Путь к странице сообщений пользователя | Указывается путь к странице сообщений пользователя. |

### Права на доступ

Вкладка позволяет настроить права на доступ к модулю для всех имеющихся в системе групп пользователей.

| Поле | Описание |
| --- | --- |
| По умолчанию | Уровень доступа групп пользователей, для которых установлено право "по умолчанию". |
| [группа\_пользователей] | Изменение настроек доступа для конкретной группы. Возможно назначение следующих прав доступа к модулю **Социальная сеть**:  * **[D] работа в публичной части без права создания групп** - просмотр страниц социальной сети с возможностью редактирования своего профиля без права создания рабочих групп; * **[K] работа в публичной части с правом создания групп** - просмотр страниц социальной сети с возможностью редактирования своего профиля с правом создания рабочих групп; * **[R] просмотр административной части** - управление социальной сетью в публичном разделе и просмотр административной части без права модификаций; * **[W] полный доступ** - полный доступ к ресурсам модуля. |
| Добавить право доступа | Ссылка, позволяющая добавить дополнительное право доступа для группы пользователей. |

**Примечание**: Уровни прав доступа модуля **Работа в публичной части с правом создания групп** и **Полный доступ** можно разделять по сайтам.
Если в портале используется многодепартаментность или в сайт менеджере многосайтовость, то администратор соцсети одного сайта (департамента) сможет иметь полный доступ или право на создание групп в рамках соцсети этого сайта, а на другом сайте он будет иметь права в соответствии с настройками групп соцсети.
