# Настройки рейтингов

Закладки

Данная форма - средство для редактирования настроек рейтингов.

  

### Настройки рейтингов

| Поле | Описание |
| --- | --- |
| Система расчета веса голоса | Устанавливается система расчета веса голоса:  * автоматическая - вес голоса будет рассчитываться [автоматически по формулам](https://dev.1c-bitrix.ru/community/blogs/rsv-dev/2386.php). * настраиваемая - вес голоса будет рассчитываться по заданным параметрам. |
| Для расчета авторитета используется рейтинг | В выпадающем списке выберите один активных рейтингов, созданных в системе, для расчета авторитета. |
| Текущий размер сообщества (пользователей) | Числовой показатель, высчитывается при пересчете авторитета. В размер сообщества включаются пользователи, которые заходили на сайт не позднее 90 дней с момента расчета и имеют право голосовать (за рейтинг или авторитет). |
| Рассчитывать нормировочный коэффициент | Выбор ручного или автоматического расчета нормированного веса голоса (варианты выбора: **Да**, **Нет**    При выборе **Нет** становится доступной смена значения коэффициента **Нормированный вес голоса**. По умолчанию - 1000. ). |
| Кол-во пользователей, за авторитет которых может голосовать пользователь в течение суток | Указывается количество пользователей. |
| Вес голоса за авторитет пользователя рассчитывается | Выберите, как будет рассчитываться вес голоса пользователя при голосовании за авторитет – по формуле (в зависимости от размера сообщества) или как один голос. |
| Кнопки голосования | |
| Использовать оценку контента | Разрешение на вывод кнопок голосования в компонентах сайта. Централизованное включение и выключение голосования. |
| Тип оценки контента | Выбор вариантов оценки:  * Мне нравится. Оценка будет производится только в положительном плане. Данную оценку можно будет "забрать" обратно. При выборе этого чекбокса становятся доступными   поля настройки текста кнопки.      Поля настройки текста кнопки:+ **Текст кнопки "Мне нравится" до голосования**   + **Текст кнопки "Мне нравится" после голосования**   + **Текст кнопки "Мне нравится" для гостей** * Нравится / Не нравится. Оценка производится как в положительную сторону, так и в отрицательную. Данную оценку можно будет "забрать" обратно. |
| Вид кнопоки | Выберите, как будет отображаться инструмент для голосования. Виды кнопок меняются в зависимости от **Типа оценки контента**. |
| Поля настроек текстов кнопки | Доступны при выборе Типа оценки **Мне нравится**. Можно задать нужный текст для кнопок голосования. |

### Право на голосование

| Поле | Описание |
| --- | --- |
| Пользователь может голосовать за свой контент | Указать может ли автор контента голосовать за собственный контент. |
| Кол-во дней отсутствия пользователя на сайте для учета в размере сообщества | Значение поля по умолчанию равно 90 дней. Вы можете его переопределить. |
| Автоматически включать пользователей в группу | [Y\N] Разрешает автоматическое включение новых пользователей в группу при регистрации. |
| **Рейтинг** | |
| Группа, имеющая право голосовать за рейтинг | В данном поле указываются группы пользователей, которые могут голосовать за рейтинг контента. С помощью [правил обработки](/user_help/service/rating/rating_rule_list.php) вы можете настроить автоматическое включение пользователей в нужную группу по достижении определенного уровня авторитета. |
| Автоматически включать пользователей в группу | Указывается группа пользователей, в которую будут включаться/исключаться пользователи при достижения ими определенного уровня нормированных голосов. Значения диапазона голосов задаются в полях **если кол-во нормированных голосов больше или равно** и **исключать если меньше**. |
| **Авторитет** | |
| Группа, имеющая право голосовать за авторитет | Здесь указываются группы пользователей, которые могут голосовать за авторитет пользователя. С помощью [правил обработки](/user_help/service/rating/rating_rule_list.php) вы можете настроить автоматическое включение пользователей в нужную группу по достижении определенного уровня авторитета. |
| Автоматически включать пользователей в группу | Указывается группа пользователей, в которую будут включаться/исключаться пользователи при достижения ими определенного уровня нормированных голосов. Значения диапазона голосов задаются в полях **если кол-во нормированных голосов больше или равно** и **исключать если меньше**. |

### Начальные значения

| Поле | Описание |
| --- | --- |
| Начальный авторитет для нового пользователя | В данном поле укажите, какой начальный авторитет будет иметь пользователь после регистрации. По умолчанию, он будет получать 3 нормированных голоса, при стандартных настройках, это позволит ему сразу голосовать за рейтинг контента и за авторитет пользователя. |
| Установить всем пользователям начальный авторитет | В данном поле вы можете выбрать начальный авторитет для всех пользователей. Эта опция создана для проектов, которые хотят запустить рейтинги на уже развитом сообществе. На основании этой настройки всем будет присвоен "Начальный авторитет":  * равный начальному значению для новых пользователей - всем будут установлены значения, равные значению в поле "Начальный авторитет для нового пользователя" (задается в [карточке пользователя](https://dev.1c-bitrix.ru/user_help/settings/users/user_edit.php#rating)); * в зависимости от активности пользователя - всем будут установлены значения, равные активности на форумах; * начальное значение и активность пользователя - в качестве авторитета будет установлено Начальный авторитет + активность, т.е. первый + второй пункт; * не устанавливать - начальный авторитет не будет устанавливаться. Это значение настройки по умолчанию в системе. |
| **Данные рейтингов** | |
| Удалить все данные системы рейтингования | Удаляет все голосования (за пост форума и т.д.) и сами результаты рейтинга — по сути это обнуление. |
