# Создание и редактирование опроса

### Закладка Опрос

| Поле | Описание |
| --- | --- |
| Название | Название опроса. |
| Дата (DD.MM.YYYY HH:MI:SS) | Указывается период, в течении которого опрос будет активным и доступным для голосования. |
| Изображение | Вы можете загрузить со своего локального компьютера какое либо изображение в форматах **\*.bmp**, **\*.jpg**, **\*.jpeg**, **\*.jpe**, **\*.gif** или **\*.png**. |
| Описание | Описание опроса в виде:  * обычного текста (**Text**); * **HTML** в виде html кодов. * **HTML** с помощью встроенного в систему **HTML** - редактора. Доступно на системах с **Internet Explorer 5.0**, **Firefox 1.0** и выше. Необходимо установить флаг **Использовать HTML-редактор** в настройках модуля **Опросы, голосования**. |

### Закладка Свойства

| Поле | Описание |
| --- | --- |
| Дата изменения | Дата изменения записи в базе данных.   Поле отображается только при редактировании опроса. |
| Количество голосов | Количество голосований по данному опросу.   Поле отображается только при редактировании опроса. |
| Активность | Флаг активности опроса. |
| Оповещать автора опроса о голосе | Выбирается способ оповещения о каждом голосовании по данному опросу:  * **Да, с помощью сообщения**; * **Да, с помощью e-mail письма**; * **Нет, не оповещать**. |
| Сортировка | Порядок сортировки опроса при выводе его в списке. |
| Группа | Группа к которой принадлежит опрос. |
| Путь | Прописывается url опроса относительно корня сайта. |
| Идентификаторы события | |
| event1 | Идентификатор типа события - **event1**. |
| event2 | Идентификатор типа события - **event2**. |
| event3 | Дополнительный параметр события - **event3**. |

### Закладка Учёт голосов

| Поле | Описание |
| --- | --- |
| Ограничение | |
| Разрешить только один голос за сессию | пользователь может проголосовать только один раз за свою сессию. |
| Разрешить один голос с одинаковым cookie | пользователь может проголосовать только один раз если используется один coockie. |
| Ограничить по IP в течение | При включении опции активизируются поля, позволяющие задать временной период в течение которого будет ограничено голование для одного IP |
| не голосовать дважды с одним ID пользователя (неавторизованным голосование запрещено) | опция не позволит проголосовать посетителю дважды с одним уникальным ID авторизованного пользователя. |
| Запретить участие пользователям, зарегистрированным после даты начала опроса | при отмеченной опции будет запрет участия в опросе для пользователей, которые зарегистрировались после даты его начала. Доступно после активации опции **не голосовать дважды с одним ID пользователя** . |
| Дополнительно | |
| Разрешить менять мнение | Разрешает пользователю изменить свой результат голосвания |
| Уровень анонимности | как отображать результаты голосования:  * **обычный** - пользователь выбирает сам отображать его голос в привязке к личности или нет. * **публичный** - все голоса отображаются в привязке к пользователю. * **анонимный** - результаты голосования отображаются без привязки к пользователям. |
