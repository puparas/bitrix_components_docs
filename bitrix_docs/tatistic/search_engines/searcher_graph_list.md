# График индексации сайта

На странице **Графики индексации сайта** (*Аналитика > Поисковики > Индексация > Графики*) отображаются графики, иллюстрирующие индексацию сайта поисковыми системами.

  

### Фильтр

| Поле | Описание |
| --- | --- |
| Период (DD.MM.YYYY) | Временной интервал, за который необходимо построить график. Это поле присутствует, даже если фильтр свернут. |
| Поисковики | В этом списке можно выбрать поисковые системы, которые следует отразить на графике. |

Чтобы построить график по заданным критериям поиска, нажмите на кнопку **Найти**.

### Контекстная панель

| Кнопка | Описание |
| --- | --- |
| Список | Нажатие этой кнопки открывает форму со списком всех поисковых систем. |
| Диаграмма | Открывает форму с [диаграммой индексации сайта](/user_help/statistic/search_engines/searcher_diagram_list.php) поисковыми системами. |
