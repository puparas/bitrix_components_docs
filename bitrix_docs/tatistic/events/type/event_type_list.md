# Список

### Описание

Форма **Типы событий** (*Аналитика > События > Типы событий > Список*) отображает список типов событий.

Тип события представляет собой заданное разработчиком сайта событие. Примерами типов событий могут
служить:

* уход на внешнюю платежную систему для оформления заказа;
* событие, происходящее при нажатии ссылки **купить** и ведущей, например, на <http://www.softkey.ru>;
* скачивание того или иного файла, происходящее при нажатии ссылки **Скачать;**
* клик по баннеру, который показывается на Вашем сайте;
* отсылка комментария на сайте;
* переход в тот или иной раздел сайта.

Практически любое действие, происходящие на сайте, можно зафиксировать как **событие** в модуле статистики.
Разработчики сайта могут получить подробное описание, как создавать события в разделе справочной системы
(*Для разработчиков > События*).

Помимо событий, происходящих на Вашем сайте, Вы можете регистрировать события, происходящие вне сайта. К числу таких событий можно отнести события **оплата заказа** либо **отмена заказа** (см. [Список событий](/user_help/statistic/events/event_list.php)).

### Фильтр

| Параметр | Описание |
| --- | --- |
| **Найти** | Поле, позволяющее найти события по их основным параметрам: **идентификатору**, **event1** или **event2**.   Это поле присутствует, даже если фильтр свернут. |
| ID\* | ID типа события в базе. |
| event1\* | Идентификатор типа события – **event1.** |
| event2\* | Идентификатор типа события – **event2.** |
| Наименование\* | Наименование типа события. |
| Описание\* | Описание типа события. |
| Первое (DD.MM.YYYY) | Интервал для поля **Первое** (дата первого события данного типа). |
| Последнее (DD.MM.YYYY) | Интервал для поля **Последнее** (дата последнего события данного типа). |
| Период (DD.MM.YYYY) | Временной интервал, за который вы хотите вывести количество событий соответствующих типов. |
| Всего событий | Интервал для поля **Всего** (общее количество событий произошедших за время ведения статистики). |
| Денежная сумма | Интервал для фильтрации суммы денег по всем событиям того или иного типа за все время (столбец **Всего**). |
 Сколько дней хранить каждое событие | Фильтр по интервалу количества дней, отводимых для хранения искомых типы событий (в случае, если это количество задано, а не используется по умолчанию). || Сколько дней хранить динамику типа события | Фильтр по интервалу количества дней, отводимых для хранения динамики искомых типы событий (в случае, если это количество задано, а не используется по умолчанию). |
| Включить в отчет по рекламе | Позволяет выбрать типы событий с определённым значением флага **Включить в отчет по рекламе**. |
| Включить в круговую диаграмму и график по умолчанию | Позволяет выбрать типы событий с определённым значением флага  **Включить в круговую диаграмму и график по умолчанию**. |

\* - для данных полей вы можете воспользоваться [специальными логическими выражениями](https://dev.1c-bitrix.ru/api_help/main/general/filter.php).

Чтобы установить фильтр по заданным критериям поиска, нажмите на кнопку **Найти**. Для отображения всех типов событий нажмите на кнопку **Отменить**.

### Контекстная панель

| Кнопка | Описание |
| --- | --- |
| Добавить | Открывает форму создания нового типа событий. |
| Группировка | Выпадающий список, в котором можно выбрать метод группировки типов событий для отображения: без группировки, с группировкой по **event1** или **event2**.  Группировка видоизменяет таблицу типов, при этом каждая строка в таблице отображает данные по всем типам событий с одинаковыми **event1** или **event2**. |
| Диаграмма | Открывает форму с [круговой диаграммой типов событий](/user_help/statistic/events/type/event_diagram_list.php). |
| Графики | Открывает форму с [графиками типов событий](/user_help/statistic/events/type/event_graph_list.php). |
| Настроить | Позволяет перейти к диалогу настройки внешнего вида отчетной формы. |
| Excel | Экспортирует данные из таблицы в формат MS Excel. |

### Типы событий

| Поле | Описание |
| --- | --- |
| Колонка флажков | Поле предназначено для выбора событий, к которым предполагается применить какое-либо действие из панели действий, расположенной ниже таблицы кампаний.  Эта колонка присутствует только в несгруппированной таблице. |
| Меню действий | Нажатие на кнопку действий в строке конкретной кампании отображает меню, содержащее команды:  * **Изменить** - открывает форму редактирования этого типа события; * **Очистить** - обнуляет всю статистику по данному типу события; * **Удалить** - удаляет тип события; * **График** - открывает форму [График типов событий](/user_help/statistic/events/type/event_graph_list.php) с фильтром, установленным на данный тип события; * **Динамика** - открывает форму [Динамика типа события по дням](/user_help/statistic/events/type/event_dynamic_list.php) с фильтром, установленным на данный тип события; * **Кампании** - открывает форму [Сравнительный анализ рекламных кампаний](/user_help/statistic/advertising_campaigns/adv_analysis.php) с фильтром, установленным на данный тип события.  Эта колонка присутствует только в несгруппированной таблице. |
| ID | Идентификатор типа события.  Эта колонка присутствует только в несгруппированной таблице. |
| Наименование | Наименование типа события.  Эта колонка присутствует только в несгруппированной таблице. |
| event1 | Идентификатор типа события – **event1**.   Эта колонка присутствует, если таблица не сгруппирована или сгруппирована по **event1**. |
| event2 | Идентификатор типа события – **event2**.   Эта колонка присутствует, если таблица не сгруппирована или сгруппирована по **event2**. |
| Сегодня | Количество событий данного типа за текущий день. |
| Вчера | Количество событий данного типа за вчерашний день. |
| Позавчера | Количество событий данного типа за позавчерашний день. |
| Всего | Общее количество событий данного типа. |
| Первое | Дата первого события данного типа. |
| Последнее | Дата последнего события данного типа. |
