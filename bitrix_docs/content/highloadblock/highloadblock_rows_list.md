# Список записей Highload-блока

### Контекстная панель

| Кнопка | Описание |
| --- | --- |
| Добавить запись | Переход к форме [добавления новой записи](/user_help/content/highloadblock/highloadblock_row_edit.php). |
| Настроить | Переход к диалогу настройки внешнего вида отчетной формы. |
| Excel | Экспорт данных из отображаемой таблицы в MS Excel. |

### Список записей

| Поле | Описание |
| --- | --- |
| Колонка флажков | Поле предназначено для выбора записей, к которым предполагается применить какое-либо правило или действие. |
| Действия | Действия над записями Highload-блока.:  * **Изменить** - [изменение](/user_help/content/highloadblock/highloadblock_row_edit.php) параметров записи; * **Копировать** - добавление новой записи путем копирования параметров текущей; * **Удалить** - удаление записи. |
| ID | Идентификатор записи Highload-блока. |
| <*поле\_блока*> | Значение поля Highload-блока. |
