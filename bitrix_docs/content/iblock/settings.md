# Информационные блоки. Настройки модуля

### Закладка "Кеширование"

| Поле | Описание |
| --- | --- |
| **Тегированный кеш** | |
| Сбрасывать кеш при наступлении даты начала/конца активности элементов инфоблоков | При нажатии кнопки **Добавить** откроется окно со списком существующих инфоблоков. Для выбранных инфоблоков будет происходить сброс кеша при наступлении [даты начала или конца активности](https://dev.1c-bitrix.ru/user_help/content/iblock/iblock_element_edit.php#element) их элементов. |
| Интервал проверки | Указывается интервал проверки наступления даты начала/конца активности элементов инфоблоков. Доступны варианты: 1, 3, 6, 12, 24 часа, либо есть возможность указать интервал вручную (в секундах). |
