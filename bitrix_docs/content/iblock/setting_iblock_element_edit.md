# Настройка внешнего вида формы редактирования

### Работа с полями и вкладками

Для добавления нужного поля (вкладки) в форму редактирования элемента необходимо перенести его из поля **Доступные поля** (**Доступные вкладки**) в **Выбранные поля** (**Вкладки**).

Для удаления ненужного поля (вкладки) с формы редактирования элемента достаточно перенести его из поля **Выбранные поля** (**Вкладки**) в **Доступные поля** (**Доступные вкладки**).

Используя этот механизм поля можно перемещать с вкладки на вкладку.
