# Местоположения 1.0

В данной главе описаны местоположения устаревшей схемы 1.0, которые доступны в более ранних версиях системы (версия модуля Интернет-магазин ниже, чем 14.10.1), либо в системе не осуществлялся переход на [местоположения 2.0](/user_help/store/sale/settings/location2/index.php).
