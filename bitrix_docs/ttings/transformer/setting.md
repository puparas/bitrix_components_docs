# Настройки модуля

Форма настроек модуля Конвертер файлов (*Настройки > Настройки продукта > Настройки модулей > Конвертер файлов*) предназначена для задания параметров подключения серверной части модуля.

| Поле | Описание | Комментарий |
| --- | --- | --- |
| Путь для добавления команд на конвертацию файлов | Указывается URL серверной части (модуль transformercontroller) |  |
| Публичный адрес сайта | Укажите URL по которому портал доступен из сети. |  |
| Режим отладки | Чекбокс включения режима отладки | Если включен, то данные о запросах и отправляемых командах будут писаться в файл `/bitrix/modules/transformer.log` |
| Время установления соединения с контроллером, секунды | Параметр установления соединения с серверной частью (модуль transformercontroller) на открытие сокета и установление соединения. |  |
| Время ожидания ответа от контроллера, секунды | Параметр установления соединения с серверной частью (модуль transformercontroller) на ожидание ответа от неё. |  |
