# Создание и редактирование валюты

Закладки

Данная форма служит для создания новой и изменения параметров созданной валюты.

  

### Контекстная панель

| Кнопка | Описание |
| --- | --- |
| Список | Переход на страницу со списком валют. |
| Создать | Переход к форме создания новой валюты. |
| Удалить | Удаление редактируемой валюты. |

### Валюта

| Поле | Описание |
| --- | --- |
| \*Валюта | Трёхбуквенный алфавитный (alfa-3) код валюты по международному стандарту [ISO 4217](https://www.iso.org/iso-4217-currency-codes.html) (например, **RUB**, **EUR**, **USD**). |
| Базовая | Отметка о том, является ли валюта базовой. Сделать валюту базовой можно из списка валют с помощью пункта меню действий **Сделать базовой**. |
| \*Номинал | Номинал, или количество валюты, для которого устанавливается обменный курс. Обычно равен номиналу, устанавливаемому ЦБ РФ. Для большинства валют номинал будет равен 1 (например, 30.5 RUB за **1** USD). У некоторых валют он будет отличен от 1 (например, 14.25 RUB за **1000** BYR). |
| \*Курс по умолчанию | Курс валюты, который будет использоваться при конвертации валюты, если в форме "Курсы валют" для данной валюты не задан другой курс. Для базовой валюты курс должен быть равен 1. Для остальных валют курс устанавливается относительно базовой валюты. Другими словами, при добавлении валюты поставьте напротив этой валюты сумму, которую дают за нее в базовой валюте.  Например, установите курс для RUB в 1, тогда курс для USD может быть равен 30.5, а для EUR - 34.2. Подробнее о курсах валют смотрите раздел помощи [Курсы валют](/user_help/settings/currency/currency_rates.php). |
| Цифровой код | Трёхзначный цифровой (number-3) код валюты по международному стандарту [ISO 4217](https://www.iso.org/iso-4217-currency-codes.html) (например **643** для RUB, **978** для EUR, **840** для USD). |
| Индекс сортировки | Относительный "вес", влияющий на положение валюты в общем списке. |

\* - Поля, обязательные для заполнения.

### Языковые настройки

| Поле | Описание |
| --- | --- |
| Настройки валют для языков системы (English, Russian и т.д.) | |
| Название | Полное название валюты для данного языка. |
| Шаблон формата | Шаблон, по которому суммы в данной валюте будут выводиться на экран.   При выборе шаблона вывода в нижеследующие поля будут подставлены значения по умолчанию. При необходимости вы можете изменить значения этих полей. |
| \*Строка формата для вывода валюты | Формат для вывода суммы в данной валюте на экран.  Все символы строки формата выводятся как есть, за исключением символа **#**. Символ **#** заменяется при выводе на реальную сумму (например, строка формата `# руб за 1 шт.` может быть выведена как `18 586,95 руб за 1 шт.`). Формат вывода суммы зависит от описанных ниже полей. |
| Десятичная точка при выводе | Символ-разделитель целой и дробной части числа. |
| Разделитель тысяч при выводе | Символ-разделитель тысяч целой части числа.  При выборе **Другое значение** можно указать свой вариант разделителя. **Для разработчиков**. Обратите внимание, что для обозначения символа-разделителя можно использовать шестнадцатеричное значение, например, **`xA0`**. |
| Количество десятичных знаков | Количество знаков в дробной части числа (например, количество копеек для RUB). Количество десятичных знаков дробной части цены влияет на отображение цен на сайте и в чеке. Все расчеты ведутся с параметрами округления, указанными в поле **Точность округления при расчетах** настроек модуля Интернет-магазин.  Это может привести к визуальным ошибкам при конвертации валют и применении скидок. В качестве примера рассмотрим такую ситуацию. Стоимость товара 98,01 руб. Если задать количество десятичных знаков равным 0, цена будет показана как 98 руб. Однако стоимость 100 единиц в корзине будет показана как 9801 руб. |
| В публичной части не показывать незначащие нули в дробной части цены | При отмеченной опции в публичной части сайта не будут показаны не имеющие значения нули дробной части цены. Если у вас цена 12500,00 рублей - будет отображено 12500, если у вас 12500,50 - будет отображено 12500,50. |

\* - Поля, обязательные для заполнения.
