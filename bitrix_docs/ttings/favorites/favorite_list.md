# Список избранного

### Описание

Управление избранными страницами осуществляется на странице **Избранное** (*Настройки > Избранное*) административного раздела сайта. Также на страницу со списком избранного можно перейти с помощью меню кнопки **Избранное** (*Избранное > Организовать избранное...*), расположенной на административной панели:

![](/upload/user_help/settings/favorite.png)

Список избранного предназначен для:

* просмотра списка избранного;
* редактирования избранных ссылок;
* создания и удаления избранных ссылок.

### Фильтр

Форма фильтра используется для выбора из списка избранного тех ссылок, которые соответствуют указанным условиям. Нижеследующая таблица описывает параметры, по которым может выполняться поиск.

| Поле | Описание |
| --- | --- |
| Найти | Позволяет найти избранные ссылки по основным параметрам: **Заголовок**, **Ссылка** и **ID**. Поле присутствует, даже если фильтр свернут. |
| Заголовок ссылки\* | Поиск ссылок по заголовку. |
| URL ссылки\* | Поиск избранного по URL ссылок. |
| ID\* | Идентификатор ссылки избранного. Соответствует полю **ID** списка избранного. |
| Язык интерфейса | Определяется, какому языку будут соответствовать избранные ссылки в результатах поиска:  * **(все)** - будут отображаться избранные ссылки, соответствующие всем языкам системы; * далее в выпадающем списке перечислены все языки, установленные в системе, по каждому из которых можно осуществлять поиск. |
| Дата изменения (DD.MM.YYYY) | Позволяет отобрать избранные ссылки, параметры которых были изменены в течение интересующего вас периода. Кнопка **Календарь** служит для выбора даты изменения из календаря. |
| Кем изменено\* | Позволяет отобрать избранные ссылки по пользователю, который изменил параметры избранного. |
| Кем создано\* | Позволяет отобрать избранные ссылки по пользователю, который их создал. |
| Описание\* | Позволяет отобрать избранные ссылки по их описанию. |
| Общая ссылка | Определяется, какие ссылки будут отображаться в результатах поиска:  * **(все)** - все ссылки (общие и необщие); * **Да** - общие для всех пользователей; * **Нет** - ссылки, не являющиеся общими для всех. |
| Принадлежит пользователю\* | Позволяет отобрать избранные ссылки по пользователю, который является их владельцем. |
| Требуются права на модуль | Позволяет установить один из модулей системы, права на доступ к которому требуются для перехода по избранным ссылкам. |

\* - для данных полей вы можете воспользоваться [специальными логическими выражениями](https://dev.1c-bitrix.ru/api_help/main/general/filter.php).

Для того чтобы отобрать избранные ссылки по заданным критериям поиска, нажмите на кнопку **Найти**. Для отображения всех ссылок нажмите на кнопку **Отменить**.

### Контекстная панель

| Кнопка | Описание |
| --- | --- |
| Добавить | Переход к [форме создания новой избранной ссылки](/user_help/settings/favorites/favorite_edit.php). |
| Настроить | Переход к диалогу настройки внешнего вида отчетной формы. |
| Excel | Экспорт данных из отображаемой таблицы в MS Excel. |

### Список групп пользователей

| Поле | Описание |
| --- | --- |
| Колонка флажков | Поле предназначено для выбора избранных ссылок, к которым предполагается применить какое-либо правило или действие. |
| Действие | Действия над избранной ссылкой:  * **Изменить** - переход к [форме редактирования избранной ссылки](/user_help/settings/favorites/favorite_edit.php); * **Удалить** - удаление избранной ссылки. |
| Заголовок | Заголовок избранной ссылки. Представлен в виде ссылки на избранную страницу. |
| Ссылка | Ссылка на избранную страницу. По данной ссылке осуществляется переход к [форме редактирования избранной ссылки](/user_help/settings/favorites/favorite_edit.php). |
| Сорт. | Индекс сортировки, который определяет положение ссылки в общем списке. |
| Язык | Язык интерфейса, на котором будет доступна избранная ссылка. |
| Общая | Если ссылка является общей для всех пользователей, то в поле будет указано **Да**, если нет - **Нет**. |
| Владелец | ID, логин и имя пользователя, который является владельцем ссылки. ID представлено в виде ссылки на форму редактирования параметров пользователя. |
| Модуль | Модуль, права на доступ к которому необходимы, чтобы данная ссылка была доступна. |
| Изменено | Дата и время последнего изменения записи о ссылке. |
| Кем | ID, логин и имя пользователя, изменившего параметры избранной ссылки. |
| ID | Идентификатор избранной ссылки. |
