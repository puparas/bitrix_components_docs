# Сервер XMPP

### Закладка "XMPP сервер"

На странице **XMPP сервер** (*Настройки > XMPP > Сервер XMPP*) вы можете управлять XMPP сервером (запустить, запросить, остановить, очистить кеш) и просмотреть основную информацию о нем.

| Поле | Описание |
| --- | --- |
| Сервер запущен | Отображается статус сервера: запущен (**Да**) или остановлен (**Нет**). |
| Пользователей на сайте | Отображается информация о количестве пользователей на сайте. |
| Соединений XMPP сервера | Отображается информация о количестве соединений XMPP сервера. |
