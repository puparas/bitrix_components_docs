# Создание и редактирование шаблона смс сообщения

Форма предназначена для создания нового или изменения уже существующего шаблона смс сообщения.

  

### Контекстная панель

| Кнопка | Описание |
| --- | --- |
| Список шаблонов | Переход на страницу со [списком шаблонов почтовых сообщений](/user_help/settings/settings/mail_events/sms_tamplate.php). |
| Добавить | Переход к форме создания нового почтового шаблона.\*\*\* |
| Копировать | Копирование редактируемого почтового шаблона.\*\*\* |
| Удалить | Удаление редактируемого почтового шаблона.\*\*\* |

### Закладка Параметры

| Поле | Описание |
| --- | --- |
| ID шаблона | Идентификатор шаблона. **Поле отображается при редактировании существующего почтового шаблона.** |
| \*Название события | Название события. |
| Активен | При отмеченной опции почтовый шаблон будет активным. Если шаблон не активен, то данный шаблон не будет использоваться для генерации почтовых сообщений. |
| \*Сайт | Указывается сайт (или сайты), для отправки сообщений с которого используется этот шаблон. |
| Язык | Указывается язык сообщения. При создании сообщения выбираются шаблоны, соответствующие языку события, либо не имеющие установленного языка. Для выбора доступны только языки, установленные в системе. |
| Тело сообщения\*\* | |
| \*Телефон отправителя \*\* | Телефон отправителя смс сообщения. |
| \*Телефон получателя \*\* | Телефон получателя смс сообщения. |
| Шаблон события | Текст сообщения. |
| Доступные поля | |
| Список шаблонов переменных (изменяемых данных), которые могут быть подставлены в поля сообщения, отмеченные **\*\***.     Например, в поле **Телефон отправителя** может быть использована переменная **#DEFAULT\_SENDER#** . Это значит, что в качестве телефона отправителя сообщения будет подставляться телефон, указанный в настройках **Главного модуля**    Поле **Номер отправителя по умолчанию** на закладке **Почта и СМС**. . | |

\* - поля, обязательные для заполнения.

\*\* - значения данных полей могут быть заданы с помощью шаблонов, представленных в секции **Доступные поля**.

\*\*\* - Кнопка отображается при редактировании существующего почтового шаблона.
