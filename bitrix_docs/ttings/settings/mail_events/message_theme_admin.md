# Темы оформления

**Тема** - это внешний вид сообщения, в котором определяется расположение различных элементов, художественный стиль и способ отображения текста сообщения. Включает в себя программный HTML-код, таблицы стилей и дополнительные шаблоны для отображения контента.

  

### Контекстная панель

| Кнопка | Описание |
| --- | --- |
| Добавить шаблон | Переход на страницу с [формой создания новой темы](/user_help/settings/settings/mail_events/message_theme_edit.php). |
| Загрузить шаблон | Переход к [форме загрузки (импорта) темы](/user_help/settings/settings/sites/template_load.php) в систему. |
| Настроить | Переход к диалогу настройки внешнего вида отчетной формы. |
| Excel | Экспорт данных из отображаемой таблицы в формат MS Excel. |

### Список тем

| Поле | Описание |
| --- | --- |
| Колонка флажков | Поле предназначено для выбора тем, к которым предполагается применить какое-либо правило или действие. |
| Действия | Действия, которые можно произвести над темой:  * **Изменить** - переход к [форме редактирования темы](/user_help/settings/settings/mail_events/message_theme_edit.php); * **Копировать** - копирование (дуплицирование) темы; * **Скачать** - сохранение темы; * **Удалить** - удаление темы. |
| ID | Идентификатор темы. |
| Название | Произвольное название темы. |
| Описание | Произвольное описание темы. |
| Скриншот | Отображается скриншот темы, если в закладке **Дополнительные файлы** формы создания/редактирования был загружен скриншот с именем **screen.gif**. |
| Порядок | Порядок следования тем в списке. |
