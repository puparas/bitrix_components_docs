# Почтовые шаблоны

### Описание

Страница **Почтовые шаблоны** (*Настройки > Настройки продукта > Почтовые и СМС события > Почтовые шаблоны*) служит для управления шаблонами почтовых сообщений. При возникновении на сайте события на отправку какого-либо письма, система ищет подходящий шаблон для такого письма. Если шаблон найден, то письмо отправляется в соответствии с этим шаблоном.

Шаблоны почтовых сообщений различаются по типам сообщений и сайтам, для которых шаблоны предназначены.

Каждый шаблон почтового сообщения может содержать два типа данных:

* **Неизменяемые данные (константы)** - не изменяются от письма к письму по этому шаблону. Другими словами, все получатели письма получат эти данные в таком виде, в котором они занесены в шаблон письма. Примером неизменяемых данных является подпись внизу письма "С уважением, служба поддержки".
* **Изменяемые данные (параметры)** - меняются от письма к письму в зависимости от разных условий. Другими словами, каждый получатель письма получит те данные, которые предназначены именно ему. Примером изменяемых данных является e-mail адрес получателя данного письма. Доступные для данного типа сообщения изменяемые данные перечислены в форме редактирования шаблона почтового сообщения в секции **Доступные поля**.

### Фильтр

Форма фильтра используется для выбора из списка почтовых шаблонов в соответствии с указанными условиями. Нижеследующая таблица описывает параметры, по которым может выполняться поиск.

| Поле | Описание |
| --- | --- |
| Найти | Поле позволяет найти требуемые почтовые шаблоны по наиболее типичным критериям, выбираемым в списке: **Тема**, **Отправитель**, **Получатель** или **Содержимое шаблона**. |
| ID | Поиск по идентификатору шаблона. |
| Тип почтового события | Поиск шаблона по типу почтового события, к которому он относится. |
| Дата изменения (DD.MM.YYYY) | Поиск шаблона по интервалу дат последнего изменения почтового шаблона. Интервал указывается в формате **DD.MM.YYYY**. |
| Сайт | Поиск шаблона по cайту, на котором используется шаблон. |
| Активность | Поиск шаблона по активным, неактивным или всем шаблонам. |
| От кого | Поиск шаблона по адресу отправителя почтового сообщения. |
| Кому | Поиск шаблона по адресу получателя почтового сообщения. |
| Скрытая копия (BCC) | Поиск шаблона по адресу получателя скрытой копии сообщения. |
| Тема | Поиск шаблона по теме почтового сообщения. |
| Тип тела сообщения | Поиск шаблона по типу тела сообщения: **text**, **html** или по обоим типам. |
| Тело сообщения | Поиск шаблона по содержимому тела сообщения. |

### Контекстная панель

| Кнопка | Описание |
| --- | --- |
| Добавить шаблон | Переход к [форме создания нового почтового шаблона](/user_help/settings/settings/mail_events/message_edit.php). |
| Настроить | Переход к диалогу настройки внешнего вида отчетной формы. |
| Excel | Экспорт данных из отображаемой таблицы в формат MS Excel. |

### Список почтовых шаблонов

| Поле | Описание |
| --- | --- |
| Колонка флажков | Поле предназначено для выбора почтовых шаблонов, к которым предполагается применить какое-либо правило или действие. |
| Действия | Действия, которые можно произвести над почтовым шаблоном:  * **Изменить** - переход к [форме редактирования параметров почтового шаблона](/user_help/service/support/components/message.edit.php); * **Добавить копию** - копирование почтового шаблона; * **Удалить** - удаление почтового шаблона. |
| ID | Идентификатор почтового шаблона. |
| Дата изм. | Дата и время последнего изменения шаблона. |
| Акт. | Признак активности почтового шаблона. Если шаблон не активен, то данный шаблон не будет использоваться для генерации почтовых сообщений. |
| Сайт | Идентификатор сайта, для отправки почтовых сообщений с которого используется данный шаблон. |
| Тип почтового события | Название типа почтового события, при возникновении которого будет сгенерировано почтовое сообщение на основе данного шаблона. |
| Тип почтового события | Код и название типа почтового события, при возникновении которого будет сгенерировано почтовое сообщение на основе данного шаблона. |
| Тема | Тема почтового сообщения. При генерации сообщения подставляется в поле **Тема**. |
| Отправитель | Адрес отправителя почтового сообщения. |
| Получатель | Адрес получателя почтового сообщения. |
| Скрытая копия (BCC) | Адрес получателя скрытой копии сообщения. |
| Тип тела сообщения | Формат тела сообщения: **text** или **html**. |
