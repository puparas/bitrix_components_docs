# Шаблоны СМС-сообщений

### Описание

Страница **Шаблоны СМС-сообщенй** (*Настройки > Настройки продукта > Почтовые и СМС события > Шаблоны СМС*) служит для управления шаблонами смс сообщений. При возникновении на сайте события на отправку смс сообщения, система ищет подходящий шаблон. Если шаблон найден, то смс отправляется в соответствии с этим шаблоном.

Шаблоны смс сообщений различаются по типам событий, языкам и сайтам, для которых шаблоны предназначены. Каждый шаблон сообщения может содержать два типа данных:

* **Неизменяемые данные (константы)** - не изменяются от сообщения к сообщению по этому шаблону. Другими словами, все получатели смс получат эти данные в таком виде, в котором они занесены в шаблон сообщения. Примером неизменяемых данных является сам текст сообщения.
* **Изменяемые данные (параметры)** - меняются от письма к письму в зависимости от разных условий. Другими словами, каждый получатель письма получит те данные, которые предназначены именно ему. Примером изменяемых данных является название сайта от имени которого производится рассылка. Доступные для данного типа сообщения изменяемые данные перечислены в форме редактирования шаблона почтового сообщения в секции **Доступные поля**.

### Фильтр

| Поле | Описание |
| --- | --- |
| Найти | Поле позволяет найти требуемые шаблоны по наиболее типичным критериям, выбираемым в списке: **Текст сообщения**, **Отправитель**, **Получатель**. |
| ID | Поиск по идентификатору шаблона. |
| Название события | Поиск шаблона по событию, к которому он относится. |
| Сайт | Поиск шаблона по cайту, на котором используется шаблон. |
| Язык сообщения | Поиск шаблона по языку сообщения. |
| Активен | Поиск шаблона по активным, неактивным или всем шаблонам. |
| Телефон отправителя | Поиск шаблона по телефону отправителя. |
| Телефон получателя | Поиск шаблона по телефону получателя. |
| Шаблон сообщения | Поиск шаблона тексту или его части в шаблоне сообщения. |

### Контекстная панель

| Кнопка | Описание |
| --- | --- |
| Добавить шаблон | Переход к [форме создания нового шаблона сообщения](/user_help/settings/settings/mail_events/sms_edit.php). |
| Настроить | Переход к диалогу настройки внешнего вида отчетной формы. |
| Excel | Экспорт данных из отображаемой таблицы в формат MS Excel. |

### Список шаблонов сообщений

| Поле | Описание |
| --- | --- |
| Колонка флажков | Поле предназначено для выбора почтовых шаблонов, к которым предполагается применить какое-либо правило или действие. |
| Действия | Действия, которые можно произвести над почтовым шаблоном:  * **Изменить** - переход к [форме редактирования](/user_help/settings/settings/mail_events/sms_edit.php) параметров шаблона смс сообщения; * **Добавить копию** - копирование шаблона; * **Удалить** - удаление шаблона. |
| ID | Идентификатор шаблона. |
| Название события | Название события. |
| Акт. | Признак активности шаблона. Если шаблон не активен, то он не используется для генерации смс сообщений. |
| Сайт | Идентификатор сайта, для отправки почтовых сообщений с которого используется шаблон. |
| Язык сообщения | Язык, используемый в сообщении. |
| Шаблон сообщения события | Текст шаблона сообщения. |
| Телефон отправителя | Телефон отправителя |
| Телефон получателя | Телефон получателя |
