# Создание и редактирование Региональной настройки

Форма создания/редактирования Региональной настройки предназначена для настройки параметров существующей или вновь создаваемой настройки.

### Контекстная панель

| Кнопка | Описание |
| --- | --- |
| Список | Переход на страницу со [списком региональных настроек](/user_help/settings/settings/lang_parametrs/culture_admin.php). |
| Добавить | Переход к форме создания нового настройки.   Кнопка отображается при редактировании существующей настройки. |
| Добавить копию | Копирование редактируемой настройки.   Кнопка отображается при редактировании существующей настройки. |
| Удалить | Удаление редактируемой настройки.   Кнопка отображается при редактировании существующей настройки. |

### Закладка Параметры

| Поле | Описание |
| --- | --- |
| ID | Отображается идентификатор Региональной настройки в системе (виден только при редактировании). |
| \*Название | Название Региональной настройки. |
| \*Формат даты (например: DD.MM.YYYY) | Указывается краткий формат показа даты в публичном и административном разделах сайта. Например, **DD.MM.YYYY**. Список основных форматов дат доступен на странице документации для разработчиков [Формат даты и времени](http://dev.1c-bitrix.ru/api_help/main/general/lang/format.php).| \*Формат даты и времени (например: DD.MM.YYYY HH:MI:SS) | Указывается полный формат показа даты и времени в публичном и административном разделах сайта. Например, **DD.MM.YYYY HH:MI:SS**. | | Первый день недели | В выпадающем списке выберите день недели, который в данном регионе принят как первый. | | \*Формат имени | В выпадающем списке выберите формат отображения имени, принятый в данном регионе.. | | \*Кодировка | Используемая [кодировка для данного языка](http://dev.1c-bitrix.ru/api_help/main/general/lang/code.php). | | Направление текста | Выберите направление текста, соответствующее данной культуре. | | Символьный код | Двузначный код настройки. | |

\* - поля, обязательные для заполнения.

### Закладка Форматы вывода

| Поле | Описание | Пример |
| --- | --- | --- |
| Форматы даты | | |
| Дата в числовом формате | Число, месяц и год через слеш или точку | **n/j/Y** (7/24/2020 или 24.07.2020) |
| Число, месяц (сокращенный) и год | Указывается формат короткой даты. | **j M Y** (24 июл 2020) |
| Число, месяц и год | Указывается обычный формат даты. | **j M Y** (30 Января 2019) |
| Число и месяц | Указывается формат отображения числа и месяца. | **j F** (30 Января) |
| Число и месяц (сокращённый) | Упрощённый вариант числа и месяцев. | **j F Y** (30 Янв) |
| День недели (сокращённый), число и месяц | Упрощённый вариант числа. | **D, j F** (Пт, 24 июл) |
| День недели (сокращённый), число и месяц (сокращённый) | Упрощённый вариант числа и месяца. | **D, j M** (Пт, 24 июл) |
| День недели, число и месяц | Обычный вариант числа и месяца. | **I, j F** (Пятница, 24 июля) |
| День недели, число, месяц и год | Полная дата с днем недели | **l, j F Y** (Среда, 30 Января 2019) |
| Форматы времени | | |
| Время без секунд | Часы и минуты. Если необходимо к записи добавляется модификатор типа [a](#modA) | **H:i** (10:24) |
| Время с секундами | Часы, минуты, секунды. Если необходимо к записи добавляется модификатор типа **a**. | **H:i:s** (10:24:13) |
| Значение am | Обозначение времени «до полудня» при [12-часовом формате исчисления времени.](https://ru.wikipedia.org/wiki/12-часовой_формат_времени) | 10:24 am |
| Значение pm | Обозначение времени «после полудня» при 12-часовом формате исчисления времени. | 3:15 pm |
| Формат чисел | | |
| Разделитель тысяч | Указывается знак препинания, которым разделяют тысячные разряды в числе. | 2,345,678.12 |
| Десятичный разделитель | Указывается знак препинания, которым разделяют целое число от десятых. | 5.25 |
| Десятичных разрядов после разделителя | Указывается, сколько чисел отображать после десятичного разделителя. | При значении 3: 5.254 |

### Модификаторы формата даты и времени

| Поле | Описание | Пример |
| --- | --- | --- |
| Y | Год. | 2019 |
| F | Название месяца в родительном падеже. | Января |
| f | Название месяца в именительном падеже. | Январь |
| M | Сокращенное название месяца. | Янв |
| m | Номер месяца в двузначном формате. | 01 |
| n | Номер месяца. | 1 |
| l     Маленькая латинская буква **L** | Название дня недели. | Среда |
| D | Сокращенное название дня недели. | Ср |
| j | День. | 30 |
| H | Часы для 24-часового формата. | **14**:25 |
| g | Часы для 12-часового формата. | **2**:25 |
| i | Минуты. | 14:**25** |
| s | Секунды. | 14:25:**23** |
| a | Отображение am/pm при 12-часовом формате исчисления времени. | **g:i:s a** 2:25:23 pm |
