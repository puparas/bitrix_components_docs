# Персональные настройки

Управление оформлением административного раздела осуществляется на странице **Настройка интерфейса пользователя** (*Настройки > Настройки продукта > Интерфейс > Персональные настройки*).

  

### Закладка Настройки

| Поле | Описание |
| --- | --- |
| Персональные настройки | |
| Разрешить контекстное меню в списках | При отмеченной опции будет отображено контекстное меню в списках (таблицах отчета). |
| Показывать контекстное меню по нажатию | Если отмечена опция **Разрешить контекстное меню в списках**,то необходимо выбрать один из способов вызова меню:  * по клику **Правой кнопки мыши**; * по клику **Правой кнопки мыши + клавиша Ctrl**. |
| Включить автосохранение форм | При отмеченной опции будет разрешена функция [Автосохранения](http://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=34&LESSON_ID=1833). |
| Панель инструментов | |
| Переходить в режим правки сайта без перезагрузки страницы | При отмеченной опции система будет переходить в режим правки сайта без перезагрузки страницы. При этом нагрузка на сервер возрастает, т.е. ускорение работы достигается большей нагрузкой на сервер. |
| Отображать панель редактирования страницы | При отмеченной опции на каждой странице сайта в режиме правки будет доступна панель **Изменить страницу в редакторе**, используя которую можно редактировать текущую страницу из публичной части. |
| Меню "Пуск" | |
| Производить предварительную загрузку меню | При отмеченной опции меню будет подгружаться предварительно в фоновом режиме. |
| Показывать всплывающие подсказки пунктов меню | При отмеченной опции будут отображаться всплывающие подсказки при наведении указателя мыши на пункты меню. |
| Количество часто используемых ссылок | Указывается количество ссылок на часто используемые страницы, список которых отображается в третьей секции меню **Пуск**. |
| Удалить все ссылки | По ссылке будет запущен Java-скрипт, который очистит список часто используемых ссылок. |
| Звуки | |
| Проигрывать звуки | При отмеченной опции при начале работы системы будет проигрываться звук, указанный в поле **Файл звука "Начало работы"**. |
| Файл звука "Начало работы" | Указывается путь до файла со звуком, который будет проигрываться. |
| Сообщения | |
| Выводить сообщения | При отмеченной опции будут выводиться сообщения:  * О сроке окончания техподдержки; * О состоянии модуля проактивной защиты; * О состоянии модуля монитора производительности. |
| Общие настройки | |
| Установить данные настройки по умолчанию для всех пользователей | При отмеченной опции данные настройки будут использоваться как настройки по умолчанию для всех зарегистрированных пользователей. |

\* - поля, обязательное для заполнения.

### Закладка Удаление

| Ссылка | Описание |
| --- | --- |
| Очистить все собственные настройки интерфейса | Для очистки всех собственных настроек интерфейса воспользуйтесь данной ссылкой. |
| Очистить все настройки по умолчанию | Для очистки всех настроек по умолчанию воспользуйтесь данной ссылкой. |
| Очистить персональные настройки всех пользователей | Для очистки персональных настроек всех пользователей воспользуйтесь данной ссылкой. |
