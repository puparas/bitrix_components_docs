# История входов

Страница **История входов** (*Настройки > Пользователи > История входов*) содержит список попыток входа с одного устройства. Страница открывается по двойному клику на выбранном устройстве в списке [Устройств пользователя](/user_help/settings/users/user_devices.php).

  

### Фильтр

Форма фильтр + поиск используется для фильтрации в соответствии с указанными условиями, а также для поиска по списку.

| Параметр | Описание |
| --- | --- |
| ID устройство | Идентификатор устройства. |
| Дата входа | Дата попытки входа. |
| IP-адрес | Адрес с которого производилась попытка входа. |
| Код города | Код города по геолокации с которого производился вход. |
| Код региона | Код региона по геолокации с которого производился вход. |
| Код страны | Код страны по геолокации с которого производился вход. |

Для того чтобы отобрать пользователей по заданным критериям поиска, нажмите на кнопку **Найти**. Для отображения всех групп нажмите на кнопку **Сбросить**.

### История входов

| Поле | Описание |
| --- | --- |
| ID | Идентификатор записи |
| ID устройства | Идентификатор устройства. |
| Дата входа | Дата попытки входа. |
| Город | Код города по геолокации с которого производился вход. |
| Регион | Код региона по геолокации с которого производился вход. |
| Страна | Код страны по геолокации с которого производился вход. |
| Пароль приложения | Ссылки на соответствующие записи. |
| Сохранённый пароль |
| Одноразовый пароль |
