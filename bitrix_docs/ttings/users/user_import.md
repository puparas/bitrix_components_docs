# Импорт пользователей

Импортирование пользователей в систему осуществляется на странице **Импорт пользователей** (*Настройки > Пользователи > Импорт пользователей*).

В системе **"1C-Битрикс: Управление сайтом"** в качестве **источника данных** для импорта пользователей может использоваться:

* CSV-файл;
* Active Directory / LDAP.

#### Смотрите также:

* [Импорт пользователей (учебный курс "Администратор. Базовый")](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=35&CHAPTER_ID=04533)
